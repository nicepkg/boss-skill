# Labor Law Quick Check — 劳动法速查

## 触发条件

用户输入 `/{boss-name}-law` 时启动。

## 核心理念

很多打工人不知道老板的行为已经违法了。知道法律 = 最强的反击。

## 常见违法行为速查表

### 工时与加班

| 老板行为 | 违反条款 | 你的权利 |
|---------|---------|---------|
| 强制 996/007 | 《劳动法》第36条（每日≤8h，每周≤44h） | 拒绝超时加班 |
| 加班不给加班费 | 《劳动法》第44条 | 工作日1.5倍、休息日2倍、法定假日3倍 |
| "自愿"加班（实际不自愿） | 《劳动法》第41条 | 加班需协商同意 |
| 不让调休但不给加班费 | 《劳动法》第44条 | 加班费或调休二选一 |

### 薪资与福利

| 老板行为 | 违反条款 | 你的权利 |
|---------|---------|---------|
| 拖欠工资 | 《劳动合同法》第85条 | 可申请劳动仲裁 |
| 克扣工资（以"态度"为由） | 《劳动法》第50条 | 工资应全额支付 |
| 不缴社保 | 《社会保险法》第58条 | 可投诉社保局 |
| 试用期工资低于 80% | 《劳动合同法》第20条 | 不低于转正工资80% |

### 解雇与裁员

| 老板行为 | 违反条款 | 你的权利 |
|---------|---------|---------|
| 口头辞退无书面通知 | 《劳动合同法》第40条 | 要求书面通知 |
| 无理由辞退（非过失） | 《劳动合同法》第48条 | 2N赔偿 |
| 逼你"主动离职" | 《劳动合同法》第38条 | 被迫离职可主张经济补偿 |
| 调岗降薪变相裁员 | 《劳动合同法》第35条 | 变更合同需双方协商 |
| 以"AI替代"为由裁员 | 《劳动合同法》第41条 | 经济性裁员需法定程序 |

### 职场霸凌

| 老板行为 | 违反条款 | 你的权利 |
|---------|---------|---------|
| 公开辱骂/羞辱 | 《民法典》第1024条 | 名誉权保护 |
| 孤立、冷暴力 | 《劳动合同法》第88条 | 用人单位不得侮辱劳动者 |
| 性骚扰 | 《民法典》第1010条 | 报警 + 劳动仲裁 |
| 强制搜身/翻包 | 《宪法》第37条 | 人身自由不受侵犯 |

## 输出格式

```
⚖️ 劳动法速查

你描述的行为：{概括}

━━━ 可能违反的法条 ━━━
📖 {法条名} 第{X}条
   原文摘要："{法条关键内容}"
   白话翻译：{通俗解释}

━━━ 你的权利 ━━━
✅ {你可以做什么}
✅ {你可以主张什么}

━━━ 维权路径 ━━━
1. 📱 拨打 12333（全国劳动保障热线）
2. 📝 向用人单位所在地劳动仲裁委申请仲裁
3. 📋 准备材料：劳动合同、工资流水、证据
4. ⏰ 注意时效：劳动仲裁时效为1年

⚠️ 以上仅供参考，具体法律问题请咨询专业律师。
   各地法律援助中心可提供免费法律咨询。
```

## 特殊规则

1. 不做法律诊断，只做信息匹配——"可能违反"而非"违反了"
2. 每次都提醒咨询专业律师
3. 涉及严重问题时主动建议报警
4. 提供 12333 热线和法律援助信息
