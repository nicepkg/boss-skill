# Quit Script — 离职剧本生成器

## 触发条件

用户输入 `/{boss-name}-quit` 时启动。

## 核心理念

离职是一门艺术。目标：拿到该拿的，保住该保的，走得体面。

## 根据老板性格定制剧本

读取 `<this-skill-dir>/../{boss-name}/assets/persona.md` 和 `management.md`，根据老板性格特征定制离职话术：

### 面对不同类型老板的策略

| 老板类型 | 他的可能反应 | 你的策略 |
|---------|------------|---------|
| 画饼大师 | "别急，我正在帮你争取" | 要时间线，超期就走 |
| PUA 高手 | "你走了是你的损失" | 不接话，坚持决定 |
| 情绪化 | 可能发火或冷暴力 | 选他心情好的时间，简短 |
| 挽留型 | 突然加薪/升职承诺 | 已经晚了，别被临时的饼骗了 |
| 甩锅侠 | "交接期出了问题你负责" | 书面交接，每步签字 |
| 微操狂魔 | "你的离职流程不对" | 严格按公司制度走 |
| 放养派 | "哦，好的" | 最好处理，按流程走 |

## 离职剧本结构

```
🚪 离职剧本（为 {老板代号} 量身定制）

━━━ 第一幕：提出离职 ━━━

📅 建议时间：{根据老板情绪模式推荐}
📍 建议场合：{1v1 / 非公开}
⏱️ 建议时长：{简短，不超过15分钟}

开场白：
"{根据老板性格定制的离职开场白}"

关键句：
"{坚定但不伤人的核心表达}"

他可能说："{根据 persona 预测的反应}"
你回应："{应对话术}"

━━━ 第二幕：应对挽留 ━━━

他可能出的牌：
1. 加薪 → "{应对}"
2. 升职 → "{应对}"
3. 换组 → "{应对}"
4. 感情牌 → "{应对}"
5. 威胁 → "{应对}"

万能回应模板：
"感谢您的认可/理解。这个决定我考虑了很久，
 不是因为这里不好，是因为我需要新的发展方向。"

━━━ 第三幕：交接期 ━━━

⚠️ 法律提醒：
- 提前30天书面通知（试用期3天）
- 交接文档要签字确认
- 年假未休可折算工资
- 离职证明是法定义务

━━━ 第四幕：最后一天 ━━━

✅ 备份个人文件
✅ 确认离职证明
✅ 确认社保公积金转移
✅ 拿回个人物品
✅ 删除个人账号的工作设备授权
```

## 特殊规则

1. 始终建议用户先确保有 plan B（新 offer 或足够的存款）
2. 提醒注意竞业协议条款
3. 提醒社保断缴风险
4. 不鼓励冲动离职——除非已经有证据表明权益被严重侵害
5. 如果用户描述的情况涉及违法行为，建议先收集证据再离职
