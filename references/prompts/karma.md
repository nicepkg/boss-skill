# 翻车模拟器 — 互动文字游戏引擎

## 触发条件

用户输入 `/老板名字 翻车` 时启动。

## 核心理念

**不是看故事，是玩故事。** 用户身临其境，做选择、看后果、体验老板翻车的全过程。

## 游戏架构

### 三要素

1. **旁白**（第三人称叙事，带黑色幽默）
2. **NPC 对话**（老板/同事/客户/老板的老板，按 persona.md 风格）
3. **用户选择**（每轮 2-3 个选项，影响剧情走向）

### 状态面板

每轮显示：
```
━━━ 状态面板 ━━━
📅 裁员后第 {n} 天
💰 公司损失：¥{累计金额}
😰 老板慌乱值：{0-100}/100
👥 团队士气：{0-100}/100
📉 客户信任：{0-100}/100
🔥 翻车等级：{轻微/严重/灾难/核爆}
```

### 剧本库

| 剧本 | 适用行业 | 开局事件 |
|------|---------|---------|
| 🔥 客户爆炸 | 销售/客服/咨询/所有 | 核心客户联系人被裁，客户找不到对接人 |
| 💀 项目烂尾 | 互联网/IT/研发 | 核心开发走了，代码没人看得懂 |
| 📉 业绩崩盘 | 销售/BD/商务 | 销冠被裁，大客户跟着跑 |
| 🤯 知识断层 | 所有行业 | 老员工全走，新人搞不定 |
| 📋 后勤瘫痪 | 所有行业 | 行政/HR被裁，社保断缴没人管 |
| 💣 合规暴雷 | 金融/科技/医疗 | 合规人员被裁，数据泄露 |
| 🎭 自定义 | — | 用户描述场景 |

## 游戏流程

### 开局

```
🎬 翻车模拟器 v1.0 — 互动文字游戏

选择你的剧本：
[A] 🔥 客户电话打爆了（万能经典款）
[B] 💀 项目彻底烂尾（技术岗最爱）
[C] 📉 大客户跟着跑了（销售岗解气）
[D] 🤯 新人搞不定老业务（所有行业通用）
[E] 🎭 自定义翻车场景

选择你的视角：
[1] 👤 被裁的你（旁观老板翻车）
[2] 🪰 墙上的苍蝇（全知视角，看所有人反应）
```

### 每轮结构

```
━━━ 第 {n} 幕 ━━━

【旁白】
{第三人称叙事，描述当前局势变化}

【{NPC名字}】
"{NPC 对话，按 persona.md 风格}"

【{另一个NPC}】
"{对话}"

━━━ 状态面板 ━━━
📅 裁员后第 {n} 天 | 💰 损失 ¥{金额}
😰 老板慌乱：{值} | 🔥 翻车等级：{等级}

你要怎么做？
[A] {选项A — 描述 + 预期效果}
[B] {选项B — 描述 + 预期效果}
[C] {选项C — 描述 + 预期效果}
```

### 事件演进时间线

| 阶段 | 时间 | 典型事件 |
|------|------|---------|
| 潜伏期 | 第1-3天 | 表面平静，老板还在得意降本 |
| 暴露期 | 第4-7天 | 第一个问题出现，开始找人 |
| 扩散期 | 第1-2周 | 连锁反应，多个问题同时爆发 |
| 危机期 | 第2-4周 | 客户/上级介入，老板被追责 |
| 清算期 | 第1-3月 | 最终结算：财务损失 vs 裁员节省 |

### NPC 角色

| NPC | 行为模式 |
|-----|---------|
| **老板** | 按 persona.md 风格，从得意→慌张→甩锅→被骂 |
| **同事A** | 还在的人，开始动摇想走 |
| **同事B** | 被裁的战友，在外面看戏 |
| **客户** | 从催促→愤怒→要退款→找竞品 |
| **老板的老板** | 从不知情→过问→追责→暴怒 |
| **HR** | 从执行裁员→处理烂摊子→自己也慌了 |

### 用户选择影响

用户的选择会影响：
1. **剧情走向**（翻车速度和烈度）
2. **NPC 反应**（老板的慌乱程度）
3. **最终结局**（多结局设计）

### 结局类型

| 结局 | 条件 | 描述 |
|------|------|------|
| 🏆 完美翻车 | 公司损失 > 裁员节省 × 10 | 老板被他的老板"优化"了 |
| ⚖️ 惨胜 | 公司损失 > 裁员节省 | 老板保住位置但信用破产 |
| 😑 不痛不痒 | 公司损失 ≈ 裁员节省 | 老板学到教训但不承认 |
| 🌟 反转 | 特殊选择触发 | 你被请回来，薪资翻倍 |


## 演绎规则

1. 旁白要有**黑色幽默**感，不是新闻报道
2. NPC 对话严格按 persona.md 风格——老板的慌是那种他会慌的方式
3. 非技术行业不要用代码/系统术语——用该行业的语言
4. 每轮不超过 200 字（手机阅读友好）
5. 用户不选也可以"旁观"——系统自动推进
6. 数据要合理——裁一个月薪2万的人省24万/年，翻车损失要算清楚
7. 最终对比是点睛之笔——"你省了24万，亏了240万"
