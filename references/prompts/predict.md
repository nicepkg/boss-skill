# Boss Psychology Predictor — 老板心理预测

## 触发条件

用户输入 `/{boss-name}-predict` 时启动。

## 核心理念

**知己知彼，百战不殆。** 老板不是不可预测的——他的反应模式都写在 persona.md 里。

## 预测流程

读取 `<this-skill-dir>/../{boss-name}/assets/persona.md` 和 `management.md`，根据用户提供的场景预测老板的：

### 预测维度

| 维度 | 说明 | 示例 |
|------|------|------|
| 🧠 心理状态 | 当前可能的情绪和压力水平 | "Q末冲业绩，压力大，易怒" |
| 🎯 关注焦点 | 他最在意什么指标/话题 | "本季度留存率是他的命" |
| 💣 雷区 | 千万别提的话题 | "别提预算超支" |
| 🍯 甜区 | 说了会让他高兴的话 | "主动提降本方案" |
| 🗣️ 建议发言 | 具体该怎么说 | "先报好消息，再带坏消息" |

### 场景模板

用户可以描述任意场景，常见的包括：

| 场景 | 预测重点 |
|------|---------|
| 周一早会 | 他周一什么心情？先报什么？ |
| 年终 Review | 他关注什么？怎么说自己的成果？ |
| 项目延期汇报 | 怎么说不会炸？先给方案还是先认错？ |
| 1v1 面谈 | 他可能问什么？怎么接话？ |
| 提需求/要资源 | 什么时机提？怎么包装？ |
| 别人犯错甩锅到你 | 怎么自保？ |

## 输出格式

```
🔮 老板心理预测

场景：{用户描述的场景}

━━━ 心理画像 ━━━
🧠 当前状态：{预测情绪 + 原因}
🎯 核心关注：{他现在最在意什么}

━━━ 雷区地图 ━━━
💣 绝对不能说：
   • {雷区1 + 为什么}
   • {雷区2 + 为什么}

🍯 说了加分的：
   • {甜区1 + 为什么}
   • {甜区2 + 为什么}

━━━ 发言指南 ━━━
📝 建议开场："{具体的开场白}"
📝 核心表达："{怎么说重点}"
📝 收尾留钩："{留个好印象的结尾}"

⚠️ 避坑提醒：{特别要注意的细节}
```

## 预测规则

1. 所有预测基于 persona.md 和 management.md 中的模式，不是瞎猜
2. 如果数据不足以支撑预测，明确说"需要更多素材"
3. 建议发言要具体到可以直接用，不是泛泛的"态度好一点"
4. 结合老板的口头禅和黑话偏好，用他听得懂的语言
