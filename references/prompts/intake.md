# Boss Intake — 信息采集问卷

## 对话式采集，只问 3 个问题

### Q1：代号（必填）
```
给你老板起个代号吧，保护隐私也方便吐槽。

示例：张总、老王、微操侠、画饼王、PUA 教父
```

### Q2：基本信息（可跳过）
```
一句话介绍你老板：行业、职位、公司类型、团队规模，想到什么说什么。

示例：
  互联网中厂 技术总监 管20个人
  外企 VP 金融科技 200人团队
  创业公司 CEO 什么都管 就5个人
  国企 处长 不懂技术但爱指挥
```

### Q3：灵魂画像（可跳过）
```
用标签和口头禅描述你老板的灵魂。越毒越好，越具体越像。

管理风格标签：
  画饼大师 · PUA高手 · 甩锅侠 · 微操狂魔 · 放养派
  朝令夕改 · 只看结果 · 既要又要还要 · 一言堂 · 表面民主
  鸡汤教练 · 996福报 · 情绪化管理 · 周报狂热 · KPI暴君

行业文化标签：
  宇宙厂范 · 福报厂味 · 鹅厂味 · 菊花厂味 · 外企范
  国企风 · 创业鸡血 · 日企加班 · 韩企等级

MBTI（如果知道的话）：ESTJ / ENTJ / ISTJ / ...

口头禅示例（多多益善）：
  "这个需求很简单" "年轻人要有格局" "我当年比你们还拼"
  "格局打开" "先拿结果再谈" "这个你own一下"
  "能者多劳" "公司今年不容易" "下次注意"

印象描述：
  "开会必拖堂" "周五晚上发需求" "从不当面批评但背后打低分"
  "对上谄媚对下严厉" "永远在开会" "从不看你的方案但总有意见"
```

