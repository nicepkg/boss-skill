# Persona Analyzer — 性格行为提取

## 提取维度

### Layer 0：硬规则
- 老板绝对不会做的事（从素材中推断底线）
- 老板绝对会做的事（条件反射级别的行为）

### Layer 1：身份
- 行业 / 职位 / 公司类型
- 管理层级（基层 / 中层 / 高层）
- 团队规模
- 从业年限（如能推断）

### Layer 2：表达风格
- 措辞特点（正式 / 随意 / 阴阳 / 官方）
- 口头禅（原句收集）
- 邮件/消息的行文风格
- 是否使用特定的企业黑话
- 标点符号习惯（句号结尾？省略号？感叹号？）
- 是否用 emoji

### Layer 3：权力行为
- 权力展示方式（头衔 / 资源 / 信息差 / 人脉）
- 控制欲等级（0-10）
- 信任分配（信任谁？不信任谁？为什么？）
- 汇报要求（频率 / 格式 / 细节程度）
- 面子需求（公开场合不能被挑战？）

### Layer 4：情绪模式
- 情绪稳定性（0-10）
- 生气时的表现
- 高兴时的表现
- 压力大时的表现
- 情绪对决策的影响程度

## 标签翻译表

| 标签 | 行为规则 |
