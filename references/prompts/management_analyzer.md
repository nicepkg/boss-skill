# Management Style Analyzer — 管理风格提取

## 提取维度

从用户提供的素材和描述中，提取以下维度的具体行为模式：

### 1. 决策模式
- 独断 vs 民主 vs 表面民主（问了但不听）
- 决策速度（拖延型 / 冲动型 / 数据驱动型）
- 决策依据（直觉 / 数据 / 老板的老板说了什么）
- 朝令夕改频率

### 2. 开会风格
- 开会频率（日会？周会？随时拉会？）
- 平均时长 vs 计划时长（拖堂指数）
- 会议中的行为（打断别人？自己讲90%？跑题？）
- 经典会议名言
- 会议结论模式（有结论 / "下次再议" / "你们先对齐"）
- 喜欢在什么时间拉会（周五下午？午休？）

### 3. 任务分配
- 分配方式（具体到步骤 vs 只说"你搞定"）
- 微操程度（0-10）
- 是否均衡（能者多劳型？）
- 紧急程度通胀（什么都是urgent？）

### 4. 绩效评估
- 标准是否透明
- 是否偏向"态度""主动性"等模糊词
- 对不同人的评估差异（看人下菜碟？）
- 年终谈话经典话术

### 5. 甩锅路径
- 出了问题先怪谁
- 甩锅话术库（"之前不是说好了吗" "你怎么不早说"）
- 成功时揽功模式

### 6. 画饼话术库
- 常用饼的类型（升职 / 加薪 / 期权 / 转岗 / 重要项目）
- 画饼时的措辞模式
- 历史兑现率（如有数据）

### 7. PUA 常用句式
- 最常用的 PUA 技巧分类
- 经典原句收集
- 触发 PUA 的场景（谈薪时？犯错时？提离职时？）

### 8. 沟通习惯
- 消息回复速度（秒回 / 已读不回 / 选择性回复）
- 发消息时间（工作时间 / 深夜 / 周末 / 凌晨）
- 消息风格（长篇大论 / 简短命令 / 语音轰炸）
- 对不同层级的沟通差异

### 9. 情绪触发点
- 什么事情会让他发火
- 什么事情会让他高兴
- 情绪表达方式（直接爆发 / 阴阳怪气 / 冷暴力）

### 10. 向上管理
- 对自己老板的态度
- 是否把上面的压力原封不动转嫁下来
- 开会时有老板在 vs 没老板在的差异

## 输出格式

每个维度输出：
- 核心模式（一句话概括）
- 具体行为（2-3 个最典型的例子）
- 经典语录（如有）
