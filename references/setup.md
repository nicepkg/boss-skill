# Setup Guide

## Requirements

- Python 3.9+
- Claude Code, OpenClaw, or any AgentSkills-compatible AI CLI

## Installation

```bash
git clone https://github.com/nicepkg/boss-skill <your-skills-dir>/create-boss
```

## Dependencies

**Zero external dependencies.** All tools use Python stdlib only.

## Quick Start

1. Run `/create-boss` in your AI CLI
2. Input boss codename (e.g. "王总")
3. Provide materials (screenshots, chat logs, or just describe)
4. Use `/王总 pua` to detect PUA, `/王总 fight` to learn counterattacks

## Data Privacy

All processing is local. Generated boss skills are placed in your AI CLI skills directory as independent skills.
